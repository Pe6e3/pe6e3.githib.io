Парсинг данных с сайта. Программа на WinForms

     Программа без особого практического применения. Основные задачи были:
•    Научиться парсить данные с сайта, сохранять их в объекты.
•    Использовать JSON для работы с данными, вместо БД
•    Попрактиковаться в WinForms, сделать программу приятной на вид
•    Визуализировать процесс парсинга (прогресс бар, количество объектов, названия)

      Алгоритм вкратце:
•   Программа заходит на сайт https://www.pickgamer.com/games,
•   Определяет количество страниц, на которых расположены ссылки на игры (их сайт периодически пополняется).
•   Пользователь выбирает страницы, которые хочет спарсить (по умолчанию - все)
•   После запуска парсинга программа сначала собирает url-адреса страниц каждой игры
•   Затем заходит на каждую станицу с игрой и достает данные о минимальных и рекомендуемых требованиях к игре.
•   Эти данные сохраняются в объект Game, объекты - в список games
•   После завершения парсинга games сериализуется в json формат и сохраняется в json файл на рабочий стол пользователя
•   Теперь пользователь может просматривать требования к любой игре из этого файла
•   Если у польщователя уже есть файл (например - спарсил ранее) - его можно импортировать одним нажатием кнопки (он должен лежать на рабочем столе и иметь такое же имя - AllGames.json)
