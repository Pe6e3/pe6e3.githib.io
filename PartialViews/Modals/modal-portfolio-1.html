Новостной портал с панелью администратора, MVC проект

HTML, CSS-шаблон сайта и панели администратора взят с интернета.
Что сделано самостоятельно:
• Созданы модели и репозитории для каждой сущности.
• Реализована связь с БД MS SQL через ORM с помощью Entity Framework.
• Настроена система регистрации, авторизации, различные уровни доступа.
• Пароли хранятся в захешированном виде (пароли хешируются с помощью библиотеки bcrypt).
• Реализован CRUD для постов, категорий, пунктов меню и прочих сущностей с возможностью управления из панели администратора.
• Реализована возможность добавлять комментарии пользователями от своего имени.
• С нуля реализовал генератор постов, который создает случаный(ые) пост(ы) со случайным текстом. Чтобы при тестировании сайта не создавать посты вручную
• Добавил небольшой собственный логгер, который следит, кто по каким ссылкам на сайте переходит. Записывает все в таблицу в админке. В нее попадает дата и время перехода, страница перехода, IP пользователя, его город, логин и имя, если это зарегистрированный пользователь.
• Код сайта четко структурирован, разбит на частичные представления для удобства редактирования.
• Несмотря на то, что был взят готовый шаблон, было написано много своих стилей для отдельных элементов, в некоторых местах применял Javascript.
